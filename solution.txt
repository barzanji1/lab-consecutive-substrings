create empty array result

loop i from 0 → string length
    create empty string current

    loop j from i → string length
        add character at j to current
        push current into result

return result

This solution generates all consecutive substrings of a string by using two nested loops.
The outer loop selects the starting index, and the inner loop builds substrings from that index forward.

Each substring is formed by progressively adding characters and storing them in an array.

Time complexity: O(n^2)
This is because for a string of length n, we generate roughly n(n+1)/2 substring.
